Contributions and suggestions to this repository are welcome.

1. [Fork this repository](https://help.github.com/articles/fork-a-repo/).
2. Add or edit an entry in the relevant category file under [`Docs/`](Docs/) (the [README](README.md) is the index that links to each category).
3. [Submit a pull request](https://help.github.com/articles/creating-a-pull-request/).

Please make sure your pull request follows these guidelines:

- **Search existing entries and open pull requests first**, as your suggestion may be a duplicate.
- Make an individual pull request for *each* suggestion.
- Match the format of the section you are editing: a table row with a `Tool`, `Description`, `License`, and `Pricing` column, using the existing license and pricing badges (see [How to read the entries](README.md#how-to-read-the-entries)).
- Keep descriptions concise and neutral, avoiding marketing language.
- Follow the existing order within the section.
- Use a direct, working link to the tool, and check your spelling, grammar, and cross-links.
- New categories, or improvements to the existing categorization, are welcome.

This is a curated index of AI tools, with an emphasis on open-source projects. Please only suggest tools you would genuinely recommend.

Thank you for your contributions!
